  </p>

<div id="starting">
  <h1>Starting</h1>

  *Clone the project and access the folder*

  ```bash
  $ git clone https://github.com/alisonsantosofc/makstore-app

  $ cd makstore-app
  ```

  *Install the dependencies*

  ```bash
  $ npm i
  ```

  *Start the project in development environment*

  ```bash
  $ npm run dev
  ```
